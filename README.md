# number_guessing game 
This is a simple Number Guessing Game made using python.

## 📌 Description
In this game, the computer randomly selects a number and the player has to guess the correct number.  
The program gives hints whether the guessed number is too high or too low until the correct number is guessed

## 🚀 Features
- Random number generation
- User input from keyboard
- Hints for higher or lower number
- Simple and beginner-friendly Python project

## ▶️ How to Run
1. Install Python on your system
2. Download or clone this repository
3. Run the file:


## bash
python number_guessing_game.py
